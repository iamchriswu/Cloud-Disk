# Cloud++
## 1.需求分析
  * 用户的登录注册 
  * 用户信息的更改
  * 用户文件上传
  * 用户文件搜索
  * 用户下载，删除，收藏文件
## 2.环境准备
  * JDK 1.8.0_181
  * tomcat 8.5
  * spring 4.2.4 RELEASE
  * springmvc 4.2.4 RELEASE
  * mybatis 3.2.8
  * bootstrap 4.1.0
  * Jquery 3.2.1
  * 文件上传、数据库连接、连接池所需jar包
    > * commons-fileupload-1.3.1.jar
    > * commons-io-2.0.1.jar
    > * mysql-connector-java-5.1.32.jar
    > * druid-1.0.9jar
